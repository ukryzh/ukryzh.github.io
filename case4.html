Разработка проектного дэшборда

В кейсе был разработан дэшборд для отслеживания проектов по внедрению ИИ-инструментов на процессах Банка

Инструментарий:

SQL SQL Server Management Studio
Power BI

Цель: инструмент для отслеживания статусов проектов по внедрению ИИ-инструментов

Результат: дэшборд в Power BI

Для построения дэшборда использовались данные, собранные на основании стратегии внедрения ИИ-инструментов на процессах Банка. (пример данных: raw_data.csv

Для удобства работы и дальнейших изменений данные были загружены в базу данных на сревере SSMS. После чего загружены в Power BI через подключение к серверу.

Построение дэшборда

1.Для вычисления нестандартных визуальных элементов (например, карточки с подсчетом количества проектов) использовался язык DAX.
2. Для столбца "Дата планируемого окончания" в таблице было добавлено условное форматирование таким образом, чтобы приближающийся дедлайн подсвечивался другим цветом
