Development of a project dashboard

As part of the case, a dashboard was developed to track the status of projects for the implementation of AI tools in the Bank's business processes.

Toolkit:

SQL (SQL Server Management Studio) Power BI

Goal: Creating a tool that allows real-time monitoring of the current status and progress of AI implementation projects.

Result: An interactive dashboard in Power BI reflecting the status of all projects within the framework of the AI ​​implementation strategy in the Bank.

Data generated based on the implementation strategy was used for visualization AI tools (data example: raw_data.csv)

The data was pre-loaded into the database on the server (SSMS) for easy updating and scaling. Then, a connection to the server was configured in Power BI for automatic loading and processing of information.

Building a dashboard:

1. To calculate non-standard visual elements, such as cards displaying the number of projects, measures written in DAX were used.

2. The table implemented a conditional formatting system for the "Planned Completion Date" column: cells with approaching deadlines are highlighted in a different color to improve visibility and ease of perception.
